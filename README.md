Alpha1.1.1_UrbanLegend - Futuristic Command Center (Blue HUD)
----------------------------------------------------------------
Quick start:
  1. Install Python 3.8+ and pygame: pip3 install pygame
  2. Unzip to your Linux files and run: python3 main.py
  3. Type natural-language commands into the input box at the bottom and press Enter.

Examples:
  - Alpha squad engage the enemy
  - All units charge
  - Drone 1 scout north
  - Tanks hold position
  - Everyone fall back to base

Files:
  - main.py         Entry point
  - units.py        Unit, Squad, Drone, Vehicle classes
  - world.py        World state & enemy AI
  - nlp_parser.py   Advanced offline NLP parser (Option B)
  - commander.py    Command executor
  - ui.py           Futuristic blue HUD UI
  - tutorial.py     Guided tutorial that runs on first launch
  - assets/         Placeholder images and sounds (blue HUD style)
  - requirements.txt  (pygame)
